# Bioinformatics project gene prediction

## 실행환경
- Visual Studio 2019 or higher

## 실행방법
1. 해당 github repository로부터 파일을 다운 혹은 git clone을 하여 파일을 생성해준다.

<p align="center"><img src="https://user-images.githubusercontent.com/69889235/207035705-63025b53-98a8-4168-a044-130ecb5cc7e9.png" width="700"></p>

2. 다운받은 Bioinformatics-project-gene-prediction 폴더 내부에서 statistical_approach 폴더 선택

<p align="center"><img src="https://user-images.githubusercontent.com/69889235/207035797-f30a98e2-18c8-48d3-a1b7-4c626cfbd171.png" width="700"></p>

3. Visual Studio 솔루션 파일인 statistical_approach.sln을 선택

<p align="center"><img src="https://user-images.githubusercontent.com/69889235/207035913-9dda16d5-ffcc-4172-91cd-1c982d51f4c9.png" width="700"></p>

4. 열린 Visual Studio에서 ctrl + F5를 통해 실행

<p align="center"><img src="https://user-images.githubusercontent.com/69889235/207035967-f7af3716-b07f-4775-9f29-acc191bde347.png" width="800"></p>
